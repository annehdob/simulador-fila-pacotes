
# Simulador de Transferência de Pacotes

Projeto desenvolvido em linguagem C para demonstrar a utilização da estrutura de dados Fila.

## Funcionalidades

- Divisão de arquivos em pacotes
- Armazenamento dos pacotes em uma fila encadeada
- Simulação visual da transmissão
- Liberação dinâmica de memória

## Conceitos aplicados

- Filas (FIFO)
- Ponteiros
- Structs
- Alocação dinâmica
- Listas encadeadas
